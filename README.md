# Palette-Picker

https://lkshamus-palette-picker.herokuapp.com/

An interactive application that allows the user to add randomized palettes to current projects. 

#### Tech-stack used
* Node.js
* JavaScript
* jQuery
* Knex/Express/Postgres database
